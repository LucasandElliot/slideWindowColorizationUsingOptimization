This is the README file of the demo program released for our SIGGRAPH Asia paper "Fast Image/Video Upsampling" by Qi Shan, Zhaorong Li, Jiaya Jia, and Chi Keung Tang. 

Distribution Version: 0.8 (Aug. 15, 2012) by Yang Xia, CSE department, the Chinese University of Hong Kong. 
This program is tested on Windows 7, but is still not guaranteed to be bug-free and work properly with all versions of Windows. 
It is for educational use ONLY.

=======================
THE USAGE
=======================

1. Execute ImageUpsampling.exe.
2. Load an image.
3. There is panel on the left, on which you set the magnification factor, iteration number, Sigma value, and the deconvolution weight.
